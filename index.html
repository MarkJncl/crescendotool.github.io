Welcome to The Crescendo Tool.

The Crescendo Tool is an open-source tool originally developed in the EU DESTECS (Design Support and Tooling for Embedded Control Software) research project. The focus is on using co-simulation to design and modelling cyber-physical systems. The tool is based on the Overture platform and Controllab's 20-sim.

Getting the Crescendo tool.

Downloads of ready-to-run versions of the Crescendo Tool; the current version is 2.0.8 (May 2014) . We have two types of package available: a completely bundled installer, containing both the Overture platform and the 20-Sim tool, and we also have Zip files that contain just the Crescendo/Overture component. Both types are available for 32-bit and 64-bit Java installations. Please use one of the links below to download the tool.
