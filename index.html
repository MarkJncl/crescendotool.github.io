Welcome to The Crescendo Tool.

The Crescendo Tool is an open-source tool originally developed in the EU DESTECS (Design Support and Tooling for Embedded Control Software) research project. The focus is on using co-simulation to design and modelling cyber-physical systems. The tool is based on the Overture platform and Controllab's 20-sim.

Getting the Crescendo tool.

Downloads of ready-to-run versions of the Crescendo Tool are presently hosted on the destecs.org website (and are branded as the DESTECS tool); the current version is Release 1.3.8. We expect a Crescendo-branded version to be available soon.

Getting documentation.

Documentation may be found on the download page with the ready-to-run versions of the tool.

Getting the code.

All of the Crescendo code will soon be hosted on GitHub, under the @crescendotool organisation account. Everyone is welcome to grab it, fork it, and contribute.
